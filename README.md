# 🦉 Athena

Athena is an AI-powered PDF chatbot built with Python, Streamlit, and the Google GenAI SDK.
Upload one or more PDF documents, ask questions in natural language, and receive answers grounded 
in the document content using a custom Retrieval-Augmented Generation (RAG) pipeline built without LangChain.

## Features

- 📄 Multi-PDF support
- 🔍 Semantic search with Gemini embeddings
- 🤖 Grounded AI responses
- 📚 Retrieved context viewer

## Tech Stack

- Python
- Streamlit
- Google GenAI SDK
- NumPy
- PyPDF
